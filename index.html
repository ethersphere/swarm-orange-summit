Agenda

Monday 11th June

  • Aron Fischer (Ethersphere, Colony): Storage and retrieval in swarm
  • Felix Lange (EF): Devp2p and RLPx: ethereum network layer
  • Daniel Nagy (EF, Swarm team): network topology for distributed storage
  • Fabian Vogelsteller (EF, Mist team): Swarm Integration in Mist
  • Fabio Barone (EF, Swarm team): Network Testing and Simulation Framework
  • Louis Holbrook (Jaak): pss - internode messaging protocol

Tuesday 12th June

  • Viktor Tron (EF, Swarm team): Decentralised database services
  • James Prestwich (Storj): On some overlaps between Storj and Swarm
  • Alex Leverington (Golem): An Introduction to Golem
  • Tomas Berthani (Oraclize): Oracle
  • Lefteris Karapetsas, Konrad Feldmeier (Brainbot): Raiden
  • Shahan Khatchatiurian (Consensys): Bisimuation-based Structural Summaries of Large Graphs
  • Tim Diebenschütz (Bigchain): BigchainDB
  • Zahoor Mohamed (EF, Swarm team): Swarm Fuse Demo

Wednesday 13th June

  • Aron Fischer (Ethersphere, Colony): Swarm incentive system
  • Viktor Tron (EF, Swarm team): Scalable payment systems
  • Alexey Akhunov: Many to Many Payments
  • Doug Petkanics (Livepeer): Realtime video streaming on swarm
  • Daniel Nagy (EF, Swarm team): Plausible Deniability
  • Elad Verbin: Data structures and security on Swarm
  • Julien Boutloup: Decentralized identity service for remote areas
